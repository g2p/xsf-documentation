XMPP Internet-Drafts

Last Updated: 2007-03-04

The base XMPP specifications are RFC 3920 and RFC 3921. While most XMPP extensions are defined in the XEP series published by the XMPP Standards Foundation, the following Internet-Drafts are under consideration within the IETF:

  • draft-ietf-enum-xmpp-01 (text) -- registration of an E.164 Number Mapping service for XMPP

  • draft-ietf-sieve-notify-xmpp-04 (text | HTML) -- XMPP notifications for the Sieve email filtering language

  • draft-saintandre-atompub-notify-05 (text | HTML) -- transporting Atom syndication data over the XMPP pubsub extension

  • draft-saintandre-jabberid-04 (text | HTML) -- including XMPP addresses in email headers

  • draft-saintandre-rfc3920bis-01 (text | HTML) -- proposed revisions to RFC 3920

  • draft-saintandre-rfc3921bis-01 (text | HTML) -- proposed revisions to RFC 3921

  • draft-saintandre-rfc4622bis-00 (text | HTML) -- corrections to RFC 4622

  • draft-saintandre-xmpp-interop-report-00 (text | HTML) -- a protocol feature set and protocol implementation reports related to RFC 3920 and RFC 3921

  • draft-saintandre-xmpp-simple-09 (text | HTML) -- enabling basic interoperability between XMPP and SIMPLE

  • draft-saintandre-xmpp-urn-03 (text | HTML) -- a Uniform Resource Name (URN) for XMPP extensions (approved for publication as an Informational RFC on 2007-02-27)

Older versions of these and other drafts are available in the attic (see also the CVS repository).
