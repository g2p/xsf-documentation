XMPP Internet-Drafts

Last Updated: 2007-11-08

The base XMPP specifications are RFC 3920 and RFC 3921. While most XMPP extensions are defined in the XEP series published by the XMPP Standards Foundation, the following XMPP-related Internet-Drafts are under consideration within the IETF:

  • draft-ietf-sieve-notify-xmpp-06 (text | HTML | feed) -- XMPP notifications for the Sieve email filtering language

  • draft-saintandre-rfc4622bis-01 (text | HTML

  • draft-saintandre-atompub-notify-06 (text | HTML) -- transporting Atom syndication data over the XMPP pubsub extension

  • draft-saintandre-header-im-01 (text | HTML) -- an email header field for IM URIs

  • draft-saintandre-header-pres-01 (text | HTML) -- an email header field for presence URIs

  • draft-saintandre-jabberid-07 (text | HTML) -- an email header field for Jabber IDs

  • draft-saintandre-rfc3920bis-04 (text | HTML) -- proposed revisions to RFC 3920

  • draft-saintandre-rfc3921bis-04 (text | HTML) -- proposed revisions to RFC 3921

  • draft-saintandre-rfc4622bis-01 (text | HTML | feed) -- corrections to RFC 4622 (approved for publication on 2007-11-05)

  • draft-saintandre-xmpp-feature-set-00 (text | HTML) -- a protocol feature set for interoperability testing of XMPP technologies

  • draft-saintandre-xmpp-presence-analysis-02 (text | HTML) -- an analysis of the scalability of interdomain presence federation using XMPP, for comparision with draft-ietf-simple-interdomain-scaling-analysis

  • draft-saintandre-xmpp-simple-10 (text | HTML) -- enabling basic interoperability between XMPP and SIMPLE

Older versions of these and other drafts are available in the attic (see also the source control repository).
