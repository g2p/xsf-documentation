XMPP Internet-Drafts

Last Updated: 2007-08-07

The base XMPP specifications are RFC 3920 and RFC 3921. While most XMPP extensions are defined in the XEP series published by the XMPP Standards Foundation, the following Internet-Drafts are under consideration within the IETF:

  • draft-ietf-enum-xmpp-02 (text) -- registration of an E.164 Number Mapping service for XMPP (published on 2007-08-07 as RFC 4979)

  • draft-ietf-sieve-notify-xmpp-04 (text | HTML) -- XMPP notifications for the Sieve email filtering language

  • draft-saintandre-atompub-notify-05 (text | HTML) -- transporting Atom syndication data over the XMPP pubsub extension

  • draft-saintandre-jabberid-05 (text | HTML) -- including XMPP addresses in email headers

  • draft-saintandre-rfc3920bis-03 (text | HTML) -- proposed revisions to RFC 3920

  • draft-saintandre-rfc3921bis-03 (text | HTML) -- proposed revisions to RFC 3921

  • draft-saintandre-rfc4622bis-01 (text | HTML) -- corrections to RFC 4622

  • draft-saintandre-xmpp-interop-report-00 (text | HTML) -- a protocol feature set and protocol implementation reports related to RFC 3920 and RFC 3921

  • draft-saintandre-xmpp-presence-analysis-00 (text | HTML) -- an analysis of the scalability of inter-domain presence federation using XMPP, for comparision with draft-ietf-simple-interdomain-scaling-analysis

  • draft-saintandre-xmpp-simple-10 (text | HTML) -- enabling basic interoperability between XMPP and SIMPLE

Older versions of these and other drafts are available in the attic (see also the source control repository).
