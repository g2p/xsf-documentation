Ian Paterson

Ian Paterson has been developing Jabber/XMPP applications full-time since 2001, mainly the Chatterbox web client and the BreakThru HTTP proxy server. He has also completed significant work on several XMPP extensions, especially HTTP Binding, Encrypted Sessions, and Message Archiving. Ian has been a member of the XMPP Council for 2005-2006 and 2006-2007.
