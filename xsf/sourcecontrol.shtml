XSF Source Control

All specifications, XML schemas, and registries produced by the XMPP Standards Foundation are kept under source control using the open-source Subversion system. Anonymous checkout of these files can be completed via the following Subversion command:

svn checkout svn://svn.xmpp.org:7938/xmpp

The repository can be viewed on the World Wide Web at <http://svn.xmpp.org:18080/browse/XMPP>. Changes to the repository can be tracked on the xmpp-commits email list or by means of an RSS file that is auto-generated using the Fisheye product generously donated by Cenqua.
