Registering a Public XMPP Service

In order to add your public XMPP service to the list, please send a message to the operators@xmpp.org discussion list with a subject of "public XMPP service" and including the following information:

  • The domain name of the XMPP service
  • The website address where users can find more information
  • The year when the service was launched
  • The country where the service is located
  • The CA that issued the security certificate you use for encryption
  • The server software used to run the service
  • The name and JabberID of the primary admin
  • A brief description of the service

Thank you for running a public XMPP service!
