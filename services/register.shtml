Registering a Public XMPP Service

Thank you for running a public XMPP service! In order to add your service to the list, please do the following:

  1. Subscribe to the operators@xmpp.org email list (which is a low-volume but very helpful venue for discussion among operators of XMPP services) by visiting <http://mail.jabber.org/mailman/listinfo/operators> or emailing <mailto:operators-subscribe@xmpp.org>.

  2. Send an email message to the list with a subject of "public XMPP service: [yourdomain.tld]". To make your life easier, a template for your message is provided below.

Please add my public XMPP service to the list at <http://xmpp.org/services/>. The service information is as follows:

  • The domain name of the XMPP service: [e.g., jabber.org]
  • The website address where users can find more information: [e.g., jabber.org]
  • The year when the service was launched: [e.g., 1999]
  • The country where the service is located: [e.g., IT]
  • The CA that issued the security certificate you use for encryption: [e.g., XMPP ICA]
  • The server software used to run the service: [e.g., ejabberd]
  • The name and JabberID of the primary admin: [e.g., Peter Saint-Andre / stpeter@jabber.org]
  • A brief description of the service: [e.g., the original Jabber service]
