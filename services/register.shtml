Registering a Public XMPP Service

Thank you for running a public XMPP service! In order to add your service to the list, please do the following:

  1. Subscribe to the operators@xmpp.org email list (which is a low-volume but very helpful venue for discussion among operators of XMPP services) by visiting <http://mail.jabber.org/mailman/listinfo/operators> or emailing <mailto:operators-subscribe@xmpp.org>.

  2. Send an email message to the list with a subject of "public XMPP service: [yourdomain.tld]". Please provide the following information:

    • The domain name of the XMPP service.
    • The website where users can find more information.
    • The year when the service was launched.
    • The country where the service is located.
    • The CA that issued the security certificate you use for encryption.
    • The server software used to run the service.
    • The name and JabberID of the primary admin.
    • A brief description of the service.

To make your life easier, a template for your message is provided below.

Please add my public XMPP service to the list at <http://xmpp.org/services/>. The service information is as follows:

  • domain: [e.g., jabber.org]
  • website: [e.g., jabber.org]
  • year launched: [e.g., 1999]
  • country: [e.g., IT]
  • CA: [e.g., XMPP ICA]
  • server software: [e.g., ejabberd]
  • admin name: [e.g., Peter Saint-Andre]
  • admin JID: [e.g., stpeter@jabber.org]
  • description: [e.g., the original Jabber service]
